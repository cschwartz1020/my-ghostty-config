# My Ghostty Config

Transparent dark teal theme with thick text, designed to let you read docs and watch videos behind the terminal.

## Apply to a new machine

1. Install [Ghostty](https://ghostty.org)

2. Clone this repo:
   ```bash
   git clone https://github.com/cschwartz1020/my-ghostty-config.git ~/git/my-ghostty-config
   ```

3. Create the Ghostty config directory and symlink:
   ```bash
   mkdir -p ~/.config/ghostty
   ln -sf ~/git/my-ghostty-config/config ~/.config/ghostty/config
   ```

4. Restart Ghostty.
